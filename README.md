# WeiUZ注意事项：
1）添加域名时候不要加WWW
2）回源地址不能直接写IP要写成域名
3）需要SSL的要解析下那个长的
	举例：_9546626e1e79dfa2945fa56a4b9af07e.weiuz.com
	CNAME解析到：_9546626e1e79dfa2945fa56a4b9af07e.weiuz.com.cdn.cloudflare.com
4）编辑解析需要严格按照格式【 前缀:回源地址 】英文【 , 】隔开，一级域名@需要单独设置
5）登录界面填写的是cloudflare个人账号，如果账号不存在会自动创建，请留意
